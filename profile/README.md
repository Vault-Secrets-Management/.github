## Resolving Common Vault Setup Issues

- Vault login not working? Confirm the selected Vault authentication method is enabled and that the policy allows access.  
- Applications cannot read Vault secrets? Check token scope, path permissions, and Vault access control rules.  
- Secret rotation failing? Review Vault key management settings, lease durations, and renewal permissions.  
- Local commands unavailable? Verify the Vault CLI is installed, configured, and pointed to the correct server address.  
- Container or cluster setup unstable? Recheck Vault Docker networking, Vault Kubernetes service discovery, and TLS configuration.
